The Staff Management System is a complete solution for handling employee registration, attendance tracking, and reporting. This module provides a clean interface for managing staff records and logging attendance efficiently using localStorage.

Features  
1. Staff Registration  
- Add, update, and delete staff  
- Store name, designation, phone, external ID, and ID image  
- Real-time search and filter by designation  
- View external ID image in modal  
- Responsive card-based layout  

2. Attendance Tracking  
- Check-in and check-out per staff  
- Auto-records time and location (city, state)  
- Calculates working hours  
- Daily reset and history stored by date  
- Delete wrong check-ins  
- Real-time dashboard update  

3. Export Capabilities  
- Export staff data to PDF with watermark, hotel name, and page numbers  
- Export attendance data to CSV for selected date  
- Filter attendance by date  
- Download clean, formatted reports  

Dependencies  
- jsPDF: For PDF generation  
- Font Awesome: For icons  
- Google Fonts: For typography  

Key Features  
- Responsive layout for all screens  
- Real-time clock on topbar  
- Uses localStorage for all data  
- Printable, professional PDF export  
- CSV download for attendance reports  

Usage  
Staff Management:  
- Add staff with required details  
- Search or filter staff  
- Edit or delete as needed  
- Export to PDF  

Attendance:  
- Mark check-in/check-out  
- Auto-captures time & location  
- Export selected date to CSV  
- Delete wrong entries  

Integration  
This module is part of the RECORDS Restaurant Management System and can be integrated with other modules like billing, menu, and inventory.
